# Gradle Plugin Template
Barebones template to help start new Gradle Plugin projects.

## Contains
- Template plugin
- Composite Build with Test Project
- JUnit Testing
- Git-tag based versioning
- Lombok

## Included Examples
- Creating a basic plugin
- Configuring a plugin id
- Testing the plugin
